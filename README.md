This repository contains a filtered dataset used for an Applied AI coursework project on banking complaint risk classification.

## Dataset
The file `complaints_filtered.csv` is a filtered subset of the **Consumer Financial Protection Bureau (CFPB) Consumer Complaint Database**.

- Original data source : Consumer Financial Protection Bureau (CFPB)
- Dataset URL : https://www.consumerfinance.gov/data-research/consumer-complaints/
- Year accessed : 2025

The dataset was filtered locally to retain :
- Consumer complaint narrative text
- Complaint issue category

This filtered file is provided to support reproducibility of the coursework notebook and does not represent original data collection.

## Usage
The dataset is automatically downloaded by the Jupyter notebook if it is not already present in the working directory.
